## 📌 Project Overview
- Built a machine learning model to classify news as Fake or Real  
- Applied text preprocessing and feature extraction techniques  
- Converted textual data into numerical format using TF-IDF  
- Trained a classification model and evaluated its performance

## ⚙️ Model Details
- Model Used: Logistic Regression  
- Feature Extraction: TF-IDF Vectorization  
- Achieved Accuracy: ~90–95%  

---

## 💡 How It Works
1. Input news text  
2. Clean and preprocess the text  
3. Convert text into numerical features using TF-IDF  
4. Model predicts whether the news is Fake or Real
